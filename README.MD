# Microsoft Intune Management Scripts

## 🎯 Overview

This repository contains a collection of PowerShell scripts and tools focused on Microsoft Intune administration and Windows device management. Most of the content was developed with the assistance of AI (“vibe coding”) to accelerate experimentation and automation within my Intune lab environment.

These tools were initially created for personal use to streamline recurring administrative tasks and address specific Intune management challenges. I’m sharing them publicly so others can benefit from the solutions and contribute to the broader Intune community.

---

## 📁 Repository Structure

| Folder | Description |
|--------|-------------|
| [Android/](Android/) | Android Enterprise and device management scripts |
| [Linux/](Linux/) | Linux device management and configuration tools |
| [Windows/](Windows/) | Windows PowerShell scripts for Intune management |
